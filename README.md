# bouncy-ninja

## Requirements

This projects depend on:

* [CMake](http://www.cmake.org/install/)
* OpenGL
* GLFW (version `3.0.4 +`)

## Running

Cmake into your preferred project file. The [`CMakeLists.txt`](CMakeLists.txt)
file has been tested to work and run on Linux (Mint) and Mac OS X.
**Compile and run!**
